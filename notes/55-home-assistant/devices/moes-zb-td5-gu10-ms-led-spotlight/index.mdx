# MOES ZB-TD5-RCW-GU10-MS

## MOES LED spot light (Zigbee)

| Type           | Vendor | Model              | Chipset | Firmware | Technology | Protocol   |
| -------------- | ------ | ------------------ | ------- | -------- | ---------- | ---------- |
| LED spot light | MOES   | ZB-TD5-RCW-GU10-MS |         | Tuya     | Zigbee     | Zigbee 3.0 |

- Friendly name: Smart LED Bulb
- Bought from: [Amazon.nl](https://www.amazon.nl/dp/B0C3726G89)
- €19.99 for 1 (March 2024)
- Dimensions: 50mm x 59mm
- Color: RGB + CCT
- Color temperature: 2200K-6500K
- Power: 4.7W
- Brightness: 345lm
- Form factor: GU10
- Input voltage: AC90-250V
- Beam angle: 36°

## Installation

### Introduction

This smart LED spotlight is Zigbee based, meaning all communication goes over the Zigbee technology
and protocol. It needs to be [paired](../../pairing/zigbee) with a Zigbee coordinator.

### Configuration

This device is plug and play.
